h1. Sukyky

Open source J2EE performance training project. Current example project has been implemented with Spring MVC.

h2. Setting up

Current implementation of the training project requires a local MySQL instance.

# Checkout the project sources

bc. git clone git@github.com:huljas/sukyky.git

# Build the project with maven

bc. cd sukyky
mvn install

# Configure the database connection in the *stock-trade/src/main/resources/db.properties*
# Create database

bc. cd stock-trade
mvn db-migration:create

# Start the example application

bc. mvn jetty:run
