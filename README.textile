h1. Sukyky

Open source J2EE performance training project implemented with Spring MVC.

h2. Setting up

The training project requires a local MySQL instance.

h5. 1) Clone

bc. git clone git@github.com:huljas/sukyky.git

h5. 2) Build

bc. cd sukyky
mvn install

h5. 3) Configure database

bc. vi stock-trade/src/main/resources/db.properties

h5. 4) Create database

bc. cd stock-trade
mvn db-migration:create

h5. 5) Run

bc. mvn jetty:run
